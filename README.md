# Women Safety Chatbot

A helpful AI-powered chatbot designed to assist women with safety tips, emergency guidance, and issue reporting.

## Features
- Provides calm, clear, and supportive responses for women in distress.
- Offers safety tips and next-step guidance in unsafe situations.
- Built using [Hugging Face Chat Assistants](https://huggingface.co/chat).
- Uses the meta-llama/Llama-3-3-70B-Instruct model.
- Publicly accessible chatbot.

## Direct Link
Access the chatbot: [Women Safety Chatbot on Hugging Face](https://hf.co/chat/assistant/6829ac224ac4fa1fdaec2fb3)

## How It Works
- The chatbot acts as a safety assistant.
- If a user feels unsafe, it guides them with actions they can take.
- Easy-to-understand, non-legal, non-medical advice.
